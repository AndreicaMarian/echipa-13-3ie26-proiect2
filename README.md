# 🍞 Brutăria Artizanală – Site de Prezentare

Site de prezentare pentru o brutărie artizanală din Timișoara, realizat cu **React + Tailwind CSS**, backend **Strapi (Headless CMS)**, găzduit pe **Netlify**.

---

---

## 📄 Pagini

| Pagină | Descriere | Layout |
|---|---|---|
| **Home** | Hero Banner, categorii, produse recomandate, blog preview, CTA | Full-width sections |
| **Produse** | Grid produse cu filtrare pe categorie și search | Sidebar filters + grid |
| **Blog** | Articole și rețete din brutărie | Card grid + featured article |
| **Despre Noi** | Single Type – povestea, misiunea, echipa, timeline | Editorial / story |
| **Contact** | Formular de contact + informații | Two-column form/info |

---

## ⚙️ Tehnologii

- **React 18** + **React Router v6**
- **Tailwind CSS 3** cu temă personalizată
- **Vite** build tool
- **Strapi** Headless CMS (backend)
- **Netlify** (hosting frontend)
- **Google Fonts**: Playfair Display, Lora, Source Sans 3

---

## 🎨 Caracteristici

- ✅ Design responsive (mobile, tablet, desktop)
- ✅ Meniu sticky desktop
- ✅ Meniu hamburger + offcanvas mobil
- ✅ Mod light/dark cu buton de toggle
- ✅ 3 fonturi web (Playfair Display, Lora, Source Sans 3)
- ✅ Search funcțional în navbar și pagina produse
- ✅ Filtrare produse pe categorie
- ✅ Formular de contact (bonus)
- ✅ Pagina 404 personalizată
