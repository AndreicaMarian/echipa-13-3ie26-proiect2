# 🍞 Brutăria Artizanală – Site de Prezentare

Site de prezentare pentru o brutărie artizanală din Timișoara, realizat cu **React + Tailwind CSS** pentru frontend, **Strapi (Headless CMS)** pentru backend și găzduit pe **Netlify**.

## 📄 Pagini (5 layout-uri diferite)

| Pagină | Descriere | Layout |
|---|---|---|
| **Acasă** | Hero Banner, statistici, categorii, produse recomandate, blog preview, CTA | Full-width sections |
| **Produse** | Grid produse cu filtrare pe categorie și căutare în timp real | Filtre + grid |
| **Blog** | Articole și rețete, cu articol recomandat (featured) | Featured + card grid |
| **Despre Noi** | Single Type – povestea, misiunea, valorile, timeline, echipă | Editorial / story |
| **Contact** | Formular de contact + informații + hartă | Two-column |

---

## ⚙️ Tehnologii

- **React 18** + **React Router v6**
- **Tailwind CSS 3** cu temă personalizată
- **Vite** – build tool
- **Strapi 5** – Headless CMS (Strapi Cloud)
- **Netlify** – hosting frontend
- **Google Fonts**: Playfair Display, Lora, Source Sans 3

---

## 🎨 Caracteristici

- ✅ Design responsive (mobile, tabletă, desktop)
- ✅ Meniu desktop sticky (rămâne fix la derulare)
- ✅ Meniu mobil hamburger + offcanvas
- ✅ Mod light/dark cu buton de toggle
- ✅ 3 fonturi web (tipografie din tema tweakcn – Amber Minimal)
- ✅ Căutare funcțională (navbar + pagina produse)
- ✅ Filtrare produse pe categorii
- ✅ Conținut real din Strapi (produse, articole, pagina Despre)
- ✅ Formular de contact (bonus)
- ✅ Pagina 404 personalizată
- ✅ Logo, search, copyright
